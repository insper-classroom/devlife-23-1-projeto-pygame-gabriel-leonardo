SHADOW OF THE NINJA

Jogo Download Desenvolvimento

Shadow of the Ninja é um jogo de plataforma desenvolvido por Leonardo Freitas e Gabriel Rodrigues durante a disciplina de Developer Life do Instituto de Ensino e Pesquisa INSPER

O objetivo do jogo é derrotar os inimigos até chegar no último nível, onde o jogador terá adquirido habilidades novas. O jogo se passa em uma floresta no meio do Japão, onde o Ninja está em busca de vingança

O jogo foi desenvolvido em Python, utilizando a biblioteca Pygame. Caso não possua a biblioteca pygame ou não saiba como jogar, clique aqui para instalar!

Para jogar, basta utilizar as teclas A e D para se mover para a esquerda e para a direita, respectivamente. Para pular, utilize a tecla ESPAÇO. Para atacar rapidamente, utilize a tecla E. Para atacar lentamente, dando mais dano, utilize a tecla Q.

Para instalar o jogo, clique no botão de instalar e siga os passos abaixo:

-Caso não tenha instalado, baixe o Python 3.10 e o VSCode;

-Já dentro do VSCode, certifique-se de que você possui a biblioteca PyGame instalada. Caso não tenha, digite o seguinte comando no terminal: pip install pygame;

-Após isso, extraia o arquivo .zip instalado, e abra-o pelo VsCode. Clique no arquivo jogo.py e aperte F5 para rodar o jogo;

-Caso deseje apenas clonar o repositório, clique aqui;

-Aproveite!
